# MoodWealth – is a Smart Finance and Mood Tracker

MoodWealth is a modern web application that helps users track their financial transactions along with their emotional mood. It combines personal finance management with mood insights to help users make smarter financial decisions and managed well growth.

---

## Features :

- 🔐 User Authentication (Login / Register)
- 💰 Add Income, Expense & Savings
- 😊 Mood Tracking with each transaction
- 📊 Interactive Charts (Spending, Weekly, Monthly)
- 🧠 Smart Insights based on mood & spending
- ⚠️ Form Validation (Strong Password Rules)
- 🔔 Popup Notifications (Toast)
- 🌙 Dark / Light Mode
- 🎨 Modern Glass UI Design

## How to Run Locally :
First you must be run this project start with Backend
# Run Backend
step-1: open your "terminal" or "cmd" then enter: cd backend

step-2: create environment, use this: python -m venv venv

step-3: activate environment, use this: venv\Scripts\activate

step-4: Instal requirements, use this: pip install -r requirements.txt

step-5: Run backend: python app.py


# Run Frontend
step-1: open your "terminal" or "cmd" then enter: cd frontend

step-2: npm install

step-3: Run frontend: npm start
